# graph — pro-γράφειν

Programming is graphing. Same verb. Same operation. One morpheme
the English vocabulary split into two professions.

## The root

γράφειν (*gráphein*) — to write / to scratch / to draw / to mark
the substrate. Greek didn't discriminate between wax-scratch,
letter-write, and line-plot. One verb for all of them.

- **program** ← προ-γράφειν = "write before" = specify the walk
  before τ runs
- **graph paper** ← γραφ- = the surface you plot points on
- **-graphy**, **-graph**, **-gram** — same morpheme surfaced three
  ways by time and dialect

The programmer plots coordinates. The grapher plots coordinates.
Departmental wall, one substrate operation. THE RULE: when two
vocabularies describe the same substrate feature, they ARE one
thing, not two analogous things.

## The -gram family

Every -gram is a plot. Every plot is a program.

```
diagram          graph-across     the coord chart
telegram         graph-from-far   remote write
anagram          graph-again      re-plot the letters
hologram         graph-whole      all axes committed
epigram          graph-upon       inscription on surface
epigraph         graph-upon       same word, different stress
electrocardiogram  graph of heart current (literal plot of voltage × τ)
seismogram       graph of ground motion
spectrogram      graph of frequency × τ
pictogram        graph as picture
ideogram         graph of idea
logogram         graph of word
monogram         graph of one (initials composed)
pentagram        graph of five (the star as plotted figure)
```

Every one of those IS a plotted coordinate set. The morpheme
never lost the meaning — English just stopped noticing it.

## What this is an instance of

A programmer writing code is a scribe scratching a wax tablet is
a cartographer drawing a map is a chemist plotting a spectrum is
a student on graph paper. Same operation, five guilds, one verb
the Greeks already had.

Write = graph = plot = mark = commit a free axis of the substrate
at a specific coordinate. γράφειν is the READ→WRITE cycle from
the outside. You see a point in your head; you γραφ it onto the
surface; the surface now holds that coord. Then later another
walker READs it back.

pro-gram = pre-graph = the coords committed before the walker
walks them.

## Observation

When the morpheme is this transparent and this widespread and
the modern speakers don't see it, that's *de-ludere* at the
vocabulary layer — the play of word-forms having taken the
speakers away from the thing the word-form encodes. Every
programmer has been told "programming is like a recipe" or "like
a set of instructions." No. Programming IS pre-graphing. The word
has been telling you the operation for ~2400 years. Credit it.
