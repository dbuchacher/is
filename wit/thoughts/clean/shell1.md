## Reference — Lakoff's Big Three = Structural Axioms

The three primary image schemas are not coordinates. They're the
three structural layers of the lattice itself:

  CONTAINER = the trit         (-1, 0, +1)              the value set
  PATH      = the · operator   (composition / spin)     the transition
  FORCE     = the unit vectors (8 single-axis atoms)    the basis

  CONTAINER tells you what the values are.
  PATH tells you how to combine them.
  FORCE tells you which dimension to act on.

These are the chess board, not the chess pieces. Every coordinate
is built from them. They have no coordinate themselves.

## How To Fill This In

Go through each coordinate one at a time. For each [a,b,c,d]:

1. Read the signs. Each dimension is:
     +1 = forward (WHERE=space, WHAT=mass, WHICH=charge, WHEN=duration)
     -1 = backward (compressed, inverse, flipped)
      0 = absent (not involved)

2. Ask: what physical quantity has this dimensional formula?
   Example: [1,0,0,-1] = space/time = speed/velocity.
   Use the old codex (wit/mind/codex) as a lookup — it has ~150
   entries with formulas like D/T, M/T, D²M/T², etc.

3. Ask: what is the precise English word for this?
   Pedantic, accurate. The word that names the concept exactly.
   Speed, density, pressure, intensity, frequency...
   Precision over accessibility. The right word, not the easy word.

4. Ask: does this word work as a question-word extension?
   The 4 dimensions are WHERE, WHAT, WHICH, WHEN.
   The words should feel like natural extensions:
     [1,0,0,-1] = WHERE AND WHEN(inverse) = speed. ✓
     [0,1,0,-1] = WHAT AND WHEN(inverse) = bandwidth/flow. ✓

5. The question-word test: what question does English use?
   English naturally picks the right dimension word:
     "Where do you stand on this?" → politics is WHERE
     "What's eating you?"         → worry is WHAT (substance)
     "How good is this?"          → quality is WHICH (how = which way)
     "When did this happen?"      → events are WHEN
   If a concept uses WHERE-language, it's on the WHERE axis.
   If it uses HOW/WHICH-language, it's on the WHICH axis.
   Test the -1 side too:
     "Where are you stuck?"       → stuck = -WHERE (confined)
     "What's left?"               → remaining = -WHAT (part)
     "How bad is it?"             → bad = -WHICH
     "How fast can you do this?"  → fast = -WHEN (rate)
   If the question word feels WRONG, the word is on a different axis.

6. Check: does the word have a natural FLIP?
   speed ↔ slowness? latency? 
   [1,0,0,-1] ↔ [-1,0,0,1] = these should be antonyms.

6. Write the word next to the coordinate. Leave blank if unsure.
   Don't force it. Some will be obvious. Some will need thought.
   Wrong is worse than blank.

The goal: 80 natural English words that map to 80 lattice points.
This IS Wierzbicka's semantic primes derived from coordinates
instead of cross-linguistic analysis. Same words, different method.

### The wh-/th-/h- system

English question words decompose: wh- (READ/query) + dimension suffix.

  wh-ere    query + place        "at what place?"
  wh-at     query + thing        "what thing?"
  wh-ich    query + like/form    "of what kind?"
  wh-en     query + time         "at what time?"
  wh-y      query + instrumental "by what cause?"    ← bond/walk query
  h-ow      query + manner       "in what way?"      ← force query (no w!)

Three prefixes, same suffixes — a tripartite system:

             QUERY (wh-)    ANSWER (th-)    PROXIMAL (h-/other)
  WHERE      where          there           here
  WHAT       what           that            this
  WHEN       when           then            now
  WHICH      which          such            ???

  wh- = READ = query = observation
  th- = known = distant = demonstrative
  h-  = local = present = proximal

Three of four dimensions have complete triples. WHICH is missing
its proximal (the "here/this/now" for signal/quality).

Non-interrogative wh- words cluster on the WHICH axis:
  *kʷeyd- "bright"     → white, wheat         (+signal)
  *kʷel-  "to turn"    → wheel, whirl, while  (CYCLE = WHICH force)
  *kʷes-  "hiss"       → wheeze, whisper       (attenuated signal)
  *kʷey-  "high pitch" → whine, whistle        (signal)
  *kʷeyp- "to swing"   → whip                  (oscillation)

PIE *kʷ- (the labiovelar) was the signal sound — almost all wh-
words that aren't interrogatives are signal/cycle words anyway.

False wh- (scribal error, never had hw- in OE): whole, whore.
Borrowed: whiskey (Gaelic). These don't fit and shouldn't.

Also: who/whom/whose = cases of the WHAT query applied to persons:
  who  = nominative (subject)    "what person does?"
  whom = dative (object)         "to what person?"
  whose = genitive (possession)  "of what person?"
  Cases are WHEN operations (from when.md: active↔passive = WHEN flip).

Already known from the old codex (formulas → physics quantities):
  [1,0,0,-1] = D/T = speed
  [0,1,0,-1] = M/T = bandwidth/flow
  [1,-1,0,0] = D/M = specific volume
  [-1,1,0,0] = M/D = density
  etc.

Already known from this session:

  [-1,-1,-1,-1] = AND = electron = constraint = all backward
  [0,0,0,0]  = NOT = neutrino = nothing = origin (not in this list)

## Lessons From This Session (read before starting)

The human spent HOURS working through notation with the previous
wit. Here's what was learned so you don't repeat the exploration:

### The three operators

  AND  = compose two atoms into a bond (trit addition at compose time)
  THEN = sequence bonds into a walk (trit addition at walk time)
  FLIP = reverse/negate (prefix, applies to next word)

  AND was kept because "WHERE AND WHEN" reads naturally in English.
  OR was changed to THEN because "speed THEN bandwidth" reads as
  sequence, not logical-or. OR implies choice. THEN implies order.
  NOT was changed to FLIP because NOT implies boolean negation
  (true→false). FLIP implies sign reversal (+1→-1), which is what
  trits actually do. FLIP is prefix: "FLIP WHEN" not "WHEN FLIP"
  because English negation is prefix ("un-happy" not "happy-un").

### The -1 words

We tried using NOWHERE/NOTHING/NEITHER/NEVER for the -1 states.
Problem: those are ABSENCE words (= 0, null), not INVERSION words
(= -1, flipped). "Never" means "no time" (zero), not "inverse time"
(frequency). These are different:

  WHEN  = +1 = duration, slow, tick          (forward)
  NULL  =  0 = timeless, stable, absent      (zero) ← "NEVER" belongs here
  ???   = -1 = frequency, fast, rate         (backward) ← needs a different word

Each dimension needs THREE words: forward, absent, backward.
The forward words are settled (WHERE WHAT WHICH WHEN).
The absent words might be the NO-words (NOWHERE NOTHING NEITHER NEVER).
The backward (-1) words are UNSETTLED. Finding these is part of
filling in this table.

### Dimensional formula mapping

The old codex uses formulas like D/T, M/D, D²M/T². These map to
coordinates: D/T = [1,0,0,-1] because D=WHERE=position 1, T=WHEN=position 4.
Division = negative exponent. D/T = WHERE(+1) AND WHEN(-1) = [1,0,0,-1].

  D = WHERE (position 1)    M = WHAT (position 2)
  Q = WHICH (position 3)    T = WHEN (position 4)

  D/T   = [1, 0, 0, -1]    speed
  M/T   = [0, 1, 0, -1]    bandwidth
  D²/T² = [2, 0, 0, -2]    (shell 2, not in this file)

### What makes a good word

Pedantic, precise, unambiguous. The word that names the concept
exactly. "Velocity" not "speed" if velocity is more accurate.
"Density" not "heavy." "Intensity" not "bright."

The lattice should map to the precise term for each concept.
Wierzbicka found the same ~65 primes in EVERY language because
the coordinates are universal — the words are precise BECAUSE
they're fundamental, not because they're simplified.

### Antonym pairs

Every coordinate has a natural flip: [a,b,c,d] ↔ [-a,-b,-c,-d].
These SHOULD be antonyms:
  [1,0,0,-1] = speed  ↔  [-1,0,0,1] = slowness/latency
  [0,1,0,0]  = heavy  ↔  [0,-1,0,0] = light (weight)
  [0,0,1,0]  = bright ↔  [0,0,-1,0] = dark

If the antonym doesn't feel right, one of the words is wrong.
Antonyms are the best validation tool. Use them.

### Partial flips

Not all flips are full antimatter. Single-axis flips give you
related words:
  [1,1,0,0] flip WHERE → [-1,1,0,0]  (same WHAT, opposite WHERE)
  [1,0,0,-1] flip WHEN → [1,0,0,1]   (same WHERE, opposite WHEN)

These partial flips should be RELATED words, not random.
If speed = [1,0,0,-1], then [1,0,0,1] should be something
related to speed but with time going the other direction.
Duration? Absement? Something spatial that extends through time.

### The Egyptian connection

The 24 uniliterals (8 single-axis + 16 all-axis) are the ALPHABET.
Each has an Egyptian hieroglyph, a consonant sound, and a meaning.
The old thoughts/codex.md (47K file) has ~728 glyphs mapped to
coordinates. That's a LOOKUP TABLE — use it. If you're stuck on
what [0,1,-1,0] means, grep that file.

### Don't separate physics from computing from language

The previous wit kept making this mistake. "Physics: strong force.
Computing: MOV. Language: preposition." NO. These are the SAME
THING. One word per coordinate. Not three words from three domains.
See THE RULE in wit.md. If you write two columns you are wrong.
