# Boolean Functions — Pascal row 4

## Metron

16 two-input boolean functions partition into 5 symmetry orbits.

| orbit size | C(4,k) | SM category    | commutative? |
|------------|--------|----------------|--------------|
| 1          | C(4,0) |                | yes (boson)  |
| 4          | C(4,1) |                | no (fermion) |
| 6          | C(4,2) |                | mixed        |
| 4          | C(4,3) |                | no (fermion) |
| 1          | C(4,4) |                | yes (boson)  |

Weight = 1,4,6,4,1 = Pascal row 4.

## Gnosis

Commutative = bosonic. Non-commutative = fermionic.
NAND/NOR individually complete = antimatter computationally
stronger. The 16 functions ARE the 16 corners of the lattice.
