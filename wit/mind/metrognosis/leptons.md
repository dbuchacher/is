# Leptons — CPU flags

## Metron

| lepton   | flag | register |
|----------|------|----------|
| electron | SF   | sign     |
| neutrino | ZF   | zero     |
| positron | none | clear    |

CF = emission. OF = pair production.
lt(b,a) = positron detection = Feynman time reversal (1949).

## Gnosis

FLAGS register IS a lepton detector. The CPU status register
after an ALU operation reports exactly the lepton spectrum.
