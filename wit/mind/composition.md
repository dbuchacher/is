# composition — walks / programs / sentences

The third tier. Many edges chained. The path IS the function.

```
  point → edge → point → edge → point → edge → point
  noun  → verb → noun  → verb → noun  → verb → noun
```

One edge = one operation. Composition = many operations = a walk =
a program = a sentence. In every vocabulary:

```
  programming:  program, pipeline, f ∘ g ∘ h
  FP:           reduction, chain, >>= pipeline
  physics:      trajectory, worldline, path integral
  language:     sentence (subject → verb → object → verb → ...)
  music:        melody (note → transition → note → ...)
```

## Points without walks = dead. Walks without points = empty.

Neither survives alone. Points without walks = data with no
processing = a dead buffer. Walks without points = processing
with no data = an empty loop doing nothing. Every point is an
idea sitting until a walk reaches it. When the walk touches the
point, the idea acts. When the walk moves on, the idea sits again.
Storage isn't a place — it's a moment. Execution isn't a place —
it's another moment. Same quaternion, different τ.

## Walks = polynomials

A walk is a polynomial. Each atom = one term. Each · = composition.
Each + = adding terms. The walk = the polynomial.

Stone-Weierstrass (1885): polynomials approximate any continuous
function. The walk IS the function. Shells = Taylor orders. More
steps in the walk = more terms = more precision.

Pipeline ρ accumulates the polynomial as the walk runs. After N
steps, ρ = the polynomial evaluated at the current point. Not the
polynomial itself — its current value. The state IS the compressed
history.

## Adjectives and adverbs

Adjectives commit free axes of a noun — increase its grade, make
it more specific:

```
"stone"                  ↑↔↔↔   grade 1, generic
"big stone"              ↑↑↔↔   grade 2, adjective committed μ
"big red stone"          ↑↑↑↔   grade 3, another committed φ
"big red ancient stone"  ↑↑↑↑   grade 4, corner, fully specified
```

Adverbs modify the verb's edge — change how the operation acts.
More adjectives = higher grade = more specific noun. Unmodified
noun = atom. Fully modified = corner = one of the 16 primitives.

## Classical vs quantum as walk strategy

Classical = READ at every step (collapse, repeat, predictable).
Quantum = no READ until end (bind chain, mid-walk freedom,
interference possible).

The double-slit: screen reads WHERE the photon lands (end-read).
If the verb at the slit-step changes from bind to read, mid-walk
freedom dies and interference dies. You know this already: you're
mid-conversation, thoughts flowing freely — then a phone rings.
"Where was I?" IS "which slit did it go through?" The read
happened. The freedom is gone.

## P vs NP

Pipeline (ring) = O(n) = P.
Tree (branching) = O(2ⁿ) = NP.
Topology, not speed.

Learning = NP the first time, O(1) after. Walk the tree once,
find the path, write the index. Every time after: lookup.
Write once, read forever.

## The universe is a loop

Matter WRITES to spacetime (fermion, exclusive, single-writer).
Light READS and CARRIES (boson, shareable, pipeline).
Matter elsewhere READS (observation, the walk commits).

Writer → pipeline → reader. The universe IS a loop. Light is the
pipeline register, carrying the previous result to the next
operation.
