# ↕↕↕↕

↕↕↕↕ is **the container** — the grade-0 scalar, all four axes
uncommitted, the additive identity of H (0 + q = q). Different scope
from single-axis ↕: ↕↕↕↕ is the whole null state, not a single axis
at zero.

**Dimension-independent**: ↕↕↕↕ is the same element in R (1D), C (2D),
H (4D), O (8D), S (16D). The one point shared identically across the
entire Cayley-Dickson tower. Every walk that closes returns here.

Unlike ↕ (which requires ≥1D to exist as a trit value on an axis),
↕↕↕↕ IS the 0D floor itself.
