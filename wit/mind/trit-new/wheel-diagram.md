# The wheel — ASCII attempts

## 8-spoke atom wheel (grade 1)

4 axes × 2 directions = 8 atoms. Opposing spokes share one trit.
Hub = ↕↕↕↕ (shared across all algebras).

```
                         τ↑
                       BEGIN
                         ●
                         │
           φ↓            │           μ↑
        CURSE            │         GROW
           ●             │            ●
             ╲           │          ╱
               ╲         │        ╱
                 ╲       │      ╱
                   ╲     │    ╱
                     ╲   │  ╱
                       ╲ │╱
   χ↓ ●━━━━━━━━━━━━━━━━━↕↕↕↕━━━━━━━━━━━━━━━● χ↑
   FALL                 hub                 RISE
                       ╱│╲
                     ╱  │  ╲
                   ╱    │    ╲
                 ╱      │      ╲
               ╱        │        ╲
             ╱          │          ╲
           ●            │           ●
        BLESS           │         SHRINK
           φ↑           │           μ↓
                        │
                        ●
                       END
                        τ↓
```

Reading:
- 4 cardinal axes: χ (WHERE), μ (WHAT), φ (WHICH), τ (WHEN)
- Each axis has ↑ and ↓ opposite spokes sharing one trit
- Hub ↕↕↕↕ = shared 0 across all Cayley-Dickson algebras
- The 8 spoke-endpoints = 8 grade-1 atoms

## Wheel within the wheel (Ezekiel's nesting)

H sits inside O sits inside S sits inside... each outer wheel
contains inner wheels as substructures. All share the hub.

```
                    ╭─────────────────────────╮
                    │                         │
                    │       S (16D)           │
                    │                         │
                    │    ╭──────────────╮     │
                    │    │   O (8D)     │     │
                    │    │              │     │
                    │    │  ╭────────╮  │     │
                    │    │  │  H     │  │     │
                    │    │  │  (4D)  │  │     │
                    │    │  │        │  │     │
                    │    │  │ ↕↕↕↕ │  │     │
                    │    │  │   •    │  │     │
                    │    │  │        │  │     │
                    │    │  ╰────────╯  │     │
                    │    │              │     │
                    │    ╰──────────────╯     │
                    │                         │
                    ╰─────────────────────────╯
          (Cayley-Dickson tower: each outer layer loses
           one algebraic property but shares the hub 0)
```

Hub `•` is the invariant point every layer holds identical.
Our universe = H. O is the host above. S is above O. Etc.

## Eyes round about — 16 half-spin corners on the rim

Ezekiel 1:18: "their rings were full of eyes round about them four."

At H-level, 16 half-spin corners (±½, ±½, ±½, ±½) sit on the unit
3-sphere SU(2). They're discrete observation-points on the wheel's
rim — 16 specific rotation-encodings, spaced symmetrically.

```
                ◉ ◉ ◉ ◉
              ◉         ◉
           ◉      ╲ │ ╱      ◉
                    ↕↕↕↕
           ◉      ╱ │ ╲      ◉
              ◉         ◉
                ◉ ◉ ◉ ◉
```

16 eyes on the rim (stylized). Each ◉ = one unit quaternion at
(±½, ±½, ±½, ±½). Each is a specific rotation state the wheel
can be in.

## Hub with 4 creatures (face symmetry)

Ezekiel: "four living creatures, each with 4 faces (man, lion, ox,
eagle)." Read as 4 axes × 4 trit-states (↑/↕/↓/+half/−half):

```
                      creature 1 (τ axis)
                         [4 faces]
                            │
                            │
  creature 4 ────── hub ↕↕↕↕ ────── creature 2 (μ axis)
  (φ axis)          [center]         [4 faces]
  [4 faces]           │
                      │
                      │
                  creature 3 (χ axis)
                     [4 faces]
```

4 creatures (axes) × 4 faces each = 16 face-configurations. Matches
16 corners of 4-cube OR 16 half-spin unit quaternions (both 2⁴).

## What spins

The wheel isn't static. "Spirit of the living creature was in the
wheels" (Ez 1:20) — the substrate's self-motion IS the rotation.
Empty loop still loops. The hub doesn't move; the spokes rotate
around it. Every rotation by θ is encoded as cos(θ/2) + sin(θ/2)·n
in the quaternion — the ½ lives in the rotation formula itself.

At H, one full rotation of the underlying spin requires 720° in
physical space. The wheel spins faster underneath than the spokes
appear to return.
