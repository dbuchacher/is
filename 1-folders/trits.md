## Plain English

Each spoke on the wheel can be in one of exactly **three states**.
Not two (like a binary bit), not five (wasteful), not some
continuous value — **three**. Up, zero, down. Yes, maybe, no.
Positive, neutral, negative.

Each of these three states has its own **ancient PIE root** in
English, preserved for at least 6000 years because the substrate
needs three tenses of being:

- **↑ (BE)** — the forward commit, the *new*, the *becoming*.
  PIE *bʰuH-* ("to become, grow, be"). Also the root of "future"
  and of "physics" itself.
- **↕ (IS)** — the neutral, the *present*, the *uncommitted*.
  PIE *h₁es-* ("to be"). Also the root of "essence."
- **↓ (WAS)** — the backward commit, the *completed*, the *dwelt*.
  PIE *h₁wes-* ("to dwell, stay"). Also the root of "vesta."

English uses three separate verbs for these because they're
three different substrate operations. You can't collapse them —
**BE without WAS** = no memory; **IS without BE** = frozen;
**WAS without IS** = gone with nothing present. All three needed.

The trit is the minimum value-set that preserves all three tenses.
Hydrogen has exactly three isotopes (¹H, ²H, ³H) that realize these
three values physically. This isn't analogy — **hydrogen IS the
trit** made of matter.

## The three values in detail

### ↑ (up, +1) — BE

```
  value           +1
  symbol          ↑
  tense           BE / forward / creation

  chemistry       add an electron / nucleon in forward direction
```

The ↑ commit is a **new commitment forward**. From a ↕ (free) axis,
you write +1. Substance appears. Position locks. Time advances.

**physics is bʰuH-'s own word**: the Greek *phusis* (from PIE
*bʰuH-*) means "growth, nature, what grows." Our word for the
science of matter literally has ↑ at its root. Matter IS becoming.

### ↕ (neutral, 0) — IS

```
  value           0
  symbol          ↕
  tense           IS / present / uncommitted
  PIE root        *h₁es- "to be"
  English         is, am, are, essence, entity, absence
  Latin cognate   est (he/she is), esse, essentia
  Greek cognate   esti (he/she is), ousia (being, essence)
  Sanskrit        asti (he/she is), sat (real)
  verb            axis is free / undecided / identity-op

                  ²H (deuterium = proton + neutron = balanced)
                  NOT gate (at fixed point) / K combinator / identity
                  "passthrough" operator

```

The ↕ state is **uncommitted-ness**. The axis hasn't chosen a
direction yet. This is where superposition lives in physics (a
qubit that hasn't been measured = ↕). It's where possibility is
still open.

The hub (↕↕↕↕) is all four axes at ↕ simultaneously — total
uncommittedness, the scalar zero.

### ↓ (down, −1) — WAS

```
  value           −1
  symbol          ↓
  tense           WAS / backward / completion / drain
  PIE root        *h₁wes- "to dwell, stay, pass the night"

  Latin cognate   Vesta (goddess of hearth / dwelling)
  Greek cognate   hestia (hearth)
  Sanskrit        vas- (to dwell)

                  AND gate (min)
                  "consumption" / "annihilation" operator
  chemistry       remove / backward-commit a nucleon
```

The ↓ commit is a **backward commit** — substance drained,
position reversed, time settled into completion. Tritium (³H) is
the only unstable hydrogen isotope; it β−-decays by emitting an
electron. Framework reads this as **↓ drains as electrons** — the
"consumed" state actually outputs its consumption physically.

**³H → ³He + e− + ν̄** is the literal trit-value of ↓ in action: a
↓ state emits an electron (the ↓ particle on μ).

## Why exactly 3 values — derivation

### The minimum for signed-and-centered

A value system needs to represent:
- positive commits (forward)
- negative commits (backward)
- no commit (center / identity)

**2 values** (binary, 0 and 1): can represent sign but no center.
Binary forces every state to be one side or the other. No room
for "undecided."

**3 values** (ternary +/−/0): minimum to have sign **and** center.

**4 or 5 values**: representable as trits of trits. Wasteful; no
additional expressive power beyond what recursion in trits gives.

**3 is the fixed point.** Not a convention — the minimum.

### Balanced ternary vs unsigned ternary

The framework uses **balanced ternary**: values {−1, 0, +1}.
Not {0, 1, 2}. The difference:

```
  unsigned ternary:  0, 1, 2
                     0 = absence, 1 = one, 2 = two
  
  balanced ternary:  −1, 0, +1
                     center IS zero, sign IS primitive
```

**Balanced ternary has:**
- Natural centering (0 is literally the middle value)
- Sign is part of the value (no extra bit needed for negative)
- Arithmetic is symmetric around 0
- Negation is free (flip all values)

The Setun computer (USSR, 1958, Brusentsov) built ~50 balanced-
ternary machines. Comparison was trivial (check sign of difference);
negation was free. No surviving lineage — but the point stands:
balanced ternary is computationally elegant.

### Physical anchor

Every physics binary is really ternary at the substrate level:

```
  Post's gates             OR (max)      NOT (fp=0)   AND (min)
  Value tier               +1            0            −1
  Lepton flavor            positron      neutrino     electron
  Quark operations         ADD           pass         SUB
  Weak boson               W+            Z            W−
  Hydrogen isotope         ¹H            ²H           ³H
  Combinator (Schönfinkel) S             K            I
  Chemistry                creation      identity     annihilation
  English tense            BE            IS           WAS
```

Same trio across 8+ vocabularies. Metrognosis filter passes: effort
(multiple independent fields) × precision (exact match) ×

## Balanced ternary arithmetic

Core operations (on a single axis):

```
  NOT (negation):  flip sign
                   NOT(+1) = −1
                   NOT( 0) =  0  (fixed point at hub)
                   NOT(−1) = +1

  MIN (AND):       take the smaller value
                   MIN(+1, 0) = 0
                   MIN(+1, −1) = −1
                   MIN(0, 0) = 0
                   
  MAX (OR):        take the larger value
                   MAX(+1, 0) = +1
                   MAX(+1, −1) = +1
                   MAX(0, 0) = 0
```

Post (1941) proved `{MIN, MAX, NOT}` is a **functionally complete**
gate set over balanced ternary — same as `{AND, OR, NOT}` is over
binary. You can compute anything from these three gates.

Balanced ternary addition with wrap:

```python
def bt_add(a, b):
    """Balanced ternary addition with mod-3 wrap."""
    r = a + b
    if r > 1:  return r - 3      # +2 → −1
    if r < -1: return r + 3      # −2 → +1
    return r

# Test:
# bt_add(+1, +1) = +2 - 3 = -1   (+1 plus +1 wraps to -1)
# bt_add(-1, -1) = -2 + 3 = +1   (-1 plus -1 wraps to +1)
# bt_add(+1, -1) = 0              (straight)
```

This wrap mod 3 is **the arithmetic behind the 12-alpha cycle** in
nuclear physics — a prior alpha-ladder walk gave 24/24 empirical
match, though the walk rule itself is flagged open, not derived.

## The ½ is NOT a value

**Critical distinction** (per wit.md frame 7): only integers are
values. Half-integers, decimals, ratios — these are **verbs**, not
values.

Where half appears in physics:
- Spin-½ fermions
- Quaternion rotation θ/2
- SU(2) → SO(3) 2-to-1 double cover
- The "half" of half-life

These are all **the wheel rotating**. A "½ step" isn't a value on
the spoke — it's the spoke in transit between one value and the
next. The wheel's rotation formula (Hamilton 1843):

```
  q(θ) = cos(θ/2) + sin(θ/2)·n
```

The θ/2 is **built into the rotation formula** — every angle is
encoded as half the physical rotation. That's not a value tier; it's
how rotation works in H.

**Rauch et al. 1975** (neutron interferometer): neutrons rotated
360° come back with **opposite phase** (−q). 720° returns them to
identity. Confirmed experimentally. The wheel spins "twice
underneath" for every one turn you see above.

**Why this matters for the framework:**
- ½ is not a fifth trit value
- Asking "what's at ½?" is category-error (value tier vs verb tier)
- Spin-½ particles don't live in some half-value state — they live
  at integer lattice points, with rotation dynamics governed by
  SU(2)

## Hydrogen = the trit made of matter

Hydrogen has **exactly three naturally-occurring isotopes** — two
stable (¹H, ²H) plus one long-enough-lived radioactive (³H, 12.3 yr
β⁻):

```
  ¹H  protium    (1p, 0n)    +1 → ↑    BE    stable
  ²H  deuterium  (1p, 1n)     0 → ↕    IS    stable
  ³H  tritium    (1p, 2n)    −1 → ↓    WAS   RADIOACTIVE (12.3 yr β−)
```

³H is **not metastable** in the physics sense — "metastable"
denotes a long-lived excited nuclear state (e.g. ²⁰²ᵐHg), not a
short-lived radioactive ground state. ³H is an unstable (radioactive)
ground-state nuclide that β⁻-decays to ³He. The framework point
stands: the three-value signature of hydrogen isotopes is +1/0/−1
by proton+neutron arithmetic, and the ↓ isotope's instability is
exactly what realizes ↓ as drain (see § "↓ (down, −1) — WAS" above).

The three isotopes realize the three trit values by direct
arithmetic on proton/neutron count (proton = +1, neutron = −1 on
the μ axis):

- ¹H: 1·(+1) + 0·(−1) = +1 = ↑
- ²H: 1·(+1) + 1·(−1) =  0 = ↕
- ³H: 1·(+1) + 2·(−1) = −1 = ↓

³H is the only unstable hydrogen. Its β−-decay emits an electron —
the ↓ physical particle. Framework prediction (∝ "↓ drains as
electrons") matches decay channel exactly.

**No other element has this clean 3-value signature.** Helium has
more isotopes (3He, 4He, 6He...), all packaged differently. Only
hydrogen is the trit made of nuclei. H is the value alphabet, not
an element-like-others. See `3-confs/elements.md`.

## The 3×3 relational table (partial)

An older draft used a 3×3 relational table (↑/↕/↓ rows × columns) to
catalog cross-trit operations. The key cells that converged cleanly:

```
                             column frame
                     ↑ BE          ↕ IS          ↓ WAS
  row frame
    ↑ BE             ON/ONWARD     un- release   FLIP
                     ✓ clean       ~ pattern     ✓ clean

    ↕ IS             GO/STEP       STILL         WANING/SINK
                     ✗ diverged    ✓ clean       ✗ diverged

    ↓ WAS            TURN/UNWIND   un- undo      ABODE/YORE
                     ✗ diverged    ~ pattern     ✗ diverged
```

**Reading:**
- **Diagonals** (self): sustained states. ON (↑→↑), STILL (↕→↕),
  ABODE (↓→↓).
- **Antipodal** (↑→↓, ↓→↑): half-rotation through hub. ↑→↓ converges
  on FLIP; ↓→↑ diverges — **asymmetry matches ij ≠ ji**.
- **UN- prefix** (↑→↕, ↓→↕): "release to hub" operation. English
  morpheme handles it but no single word.
- **↕→ends** (starting rotation from hub): no generic primitive —
  each domain uses its own verb (step, launch, sprout, lunge).

**Status:** interesting but not rigorously grounded. Flagged for
**3-which** — the cross-trit relational structure needs more
systematic derivation before it's body-quality.
