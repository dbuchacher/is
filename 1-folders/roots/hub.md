# hub — stillness, stability, dwelling

Roots naming the hub (↕↕↕↕) and its qualities: stability, permanence,
dwelling, holding. These anchor the framework's hub / stillness
concepts. If the wheel has a "stillness concept," it lives here.

---

### *steh₂-*

"to stand, be firm"

```
  → stand, state, stable, static, stasis, standard, constant,
    establish, statue, station, status, substance (sub-stare =
    "stand under"), destiny (de- + stare = "firmly established"),
    STILL, BIDE
  → Greek: stasis, histēmi
  → Sanskrit: sthita, tiṣṭhati
  → Latin: stare → stabilis, status, substantia
  → Germanic: stand, stehen, stan
  → the ↕ hub value, the un-moving, stability through rotation
```

This one root anchors: **hub**, **substance** (μ axis!), **destiny**
(stable bindings), **state** (a settled configuration), **constant**
(unchanging through time). Ten-plus independent IE language families,
6000 years, one semantic core: **stability, standing-firm, un-moving**.
That's the substrate's ↕ hub-value in the deepest layer of human
vocabulary we can reach.

### *men-*

"to remain, stay"

```
  → remain, permanent, manor, mansion, menage
  → Latin manere (to stay)
  → Greek menō (I stay)
  → the duration aspect of τ axis — hub-stable-across-time
```

Where *steh₂-* is spatial stillness (standing firm), *men-* is
temporal stillness (remaining across ticks). Different aspect of
hub-stability.

### *dhṛ-*

"to hold, carry, support" (Sanskrit)

```
  → dharma (!) — what holds things together, the substrate-law
  → Sanskrit dharati (holds)
  → Latin: firm, firmitude (related forms)
  → the substrate as what holds / supports existence
```

**Dharma** = what the substrate holds together. Close to reality but
active — reality is what IS, dharma is what HOLDS. The hub
operationalized: the place that holds everything else from falling
apart.

Related to *dʰeh₁-* (put, place) in the write cluster — both
involve setting/holding in place. *dʰeh₁-* is the active putting;
*dhṛ-* is the sustained holding.
