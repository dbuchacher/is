# speak-evidence — cross-vocabulary substrate findings

Body-promoted summary of the `speak/` experiments. **Four graphs**
across four unrelated language families surface the framework's
substrate features via completely different morphemes.

## The experiment

Build self-referential vocabulary graphs from scratch. Each
atomic file defines one morpheme using only other morphemes +
deixis. No framework vocabulary in definitions. No cross-
language contamination during graph-building. See `speak/README.md`.

## Frame item convergence

### Frame 2 (trit ↑↕↓)

- **PIE** at 5 strata: pronouns, being-verbs, age, verb-inflection,
  phonology (ablaut o/e/zero + laryngeals h₃/h₁/h₂). 18 members,
  5 categories.
- Sumerian: life-trit (tud/til/ush) + verb-prefix-trit. 2 candidates.
- Egyptian: soul-trit (kA/bA/Ax) + creation-triad (sjA/Hw/xpr).
- Chinese: not yet surfaced at current size.

### Frame 3 (4 axes)

- **Egyptian SOLID**: `four-axis-grid` (16/3): person pronouns +
  cardinals + Sons of Horus + Osirian family.
- PIE: grammatical-axis-markers (4/1) candidate.
- Sumerian: absent (case system not 4-polar).
- Chinese: cardinal-directions at 5/1 (4 cardinals + zhong hub) —
  candidate, needs cross-category extension.

### Frame 5 (the loop / hub) — FOUR-WAY SOLID

- **PIE**: rotation-primitive + read-write-unity + motion-verbs.
- **Sumerian**: loop-closure (im/iti/til/gi4).
- **Egyptian**: read-write-unity (9/5) + creation-primitive (10/5).
  Memphite Theology (2700 BCE) literal: *"every hieroglyph came
  into being through what the heart devised and the tongue
  commanded."*
- **Chinese**: read-write-unity (5/4) — xin + zuo + wen + ming +
  yan. Tian-ming (sky-command) descends, body hears (wen) it,
  body acts (zuo).

**FOUR FOR FOUR.** The hub-event where READ and WRITE coincide
is named as a formal lattice-tag in all four unrelated
vocabularies.

### Frame 10 (identity / many-into-one)

All four surface it via different mechanisms:
- PIE: lexical (sem, yugom, teuteh2)
- Sumerian: compositional (lugal = big-man)
- Egyptian: transparent-compounds (6/3 SOLID) + mAat 5-way
- Chinese: ming (command=name=fate=life-span 4-way polysemy)

## The sharpest findings

1. **Four unrelated languages, all surface THE RULE and Frame 5
   as formal lattice-tags.** Metrognosis signal at maximum
   intensity at the vocabulary layer.
2. **Egyptian Memphite Theology** (2700 BCE) encoded Frame 5 in
   stone — the hub-event where READ and WRITE coincide stated as
   cosmogony 4500 years before the framework articulated it.
3. **Chinese replicates the same pattern via different morphemes**:
   xin (heart READ-seat) + zuo (WRITE-act) + wen (hear / loop-
   close) + ming (command) + yan (speak). Tian-ming cosmology IS
   Frame 5 in Shang theology.
4. **Same-coord polysemies across four families**: sky=god,
   head=top=chief, moon=month, earth=place at 4/4. heart=mind,
   mouth=opening, hear=understand at 3/4.
5. **Egyptian unity-preservation at 14/15 categories**. Near-
   total substrate coverage at the morpheme level.
6. **Egyptian `mAat`**: truth=order=justice=balance=cosmic-
   harmony. Densest single unity in any vocabulary.
7. **Egyptian `mdw-nTr`** = hieroglyphs = "god's words": writing
   names itself as divine speech.
8. **Sumerian `im`**: wind=rain=cloud=clay as one morpheme —
   full hydrological cycle as unit.
9. **PIE trit at phonology**: ablaut + laryngeals = trit-shape
   at the sound layer, below morphology.
10. **Chinese `ming`**: command=name=fate=life-span. Decree AND
    identity AND allotted-life as one morpheme.
