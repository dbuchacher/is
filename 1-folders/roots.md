# roots — PIE anchors of the substrate (index)

Deep file for 1-folders.md. The wheel's structural features show up in
the **oldest reconstructable language humans have**: Proto-Indo-
European (PIE), reconstructed ~6000 years back from cognates across
100+ Indo-European languages. Greek, Latin, and Sumerian (language
isolate) provide additional independent anchors.

PIE roots **aren't etymology trivia** — they're evidence that
specific substrate features were named early, before Cartesian /
departmental vocabulary fractured things. A PIE root that survived
6000 years across independent language families is **substrate
signal** under THE FILTER (effort × precision × independent
convergence): high effort (centuries of comparative linguistics),
high precision (specific phonetic + semantic constraints),
independent (reconstructed without framework input).

This file is the index. Actual entries live in the `roots/`
cluster folder.

## The Metrognosis case

A root like *steh₂-* "to stand" gives English **stand, state,
stable, static, stasis, constant, substance** — plus Sanskrit
*sthita*, Greek *stasis*, Latin *stare*, German *stehen*. Ten-plus
independent IE language families, 6000 years, one semantic core:
**stability, standing-firm, un-moving**. That's the substrate's ↕
hub-value showing up in the **deepest layer of human vocabulary
we can reach**. The framework didn't invent the stability concept —
it was already named before writing.

When a framework concept has a PIE root anchor, it's not a framework
invention. It's the framework noticing what was named long ago.

The filter applies specifically:

- **Effort**: centuries of comparative linguistics. Grimm, Bopp,
  Benveniste, Mallory, hundreds of reconstructions cross-checked
  against hundreds of cognates.
- **Precision**: specific phonetic laws (Grimm's Law, etc.) +
  semantic constraints. Not fuzzy.
- **Independence**: PIE reconstruction predates the framework. No
  one reconstructed *steh₂-* to match a wheel-model. The root was
  there; the framework arrived later and noticed the fit.
- **Convergence**: a single root unites words across Sanskrit,
  Greek, Latin, Germanic, Celtic, Baltic, Slavic — seven
  independent language families per root, commonly. Plus Sumerian
  (isolate) provides cross-family convergence where it lines up.

PIE roots are **the substrate's fingerprints in the oldest human
vocabulary we can trace**. They belong in body-tier material.

## How to use this file

- Writing framework material: tag the first-use of each load-bearing
  term with its root via the V2 discipline (see `wit.md`).
- Looking up which cluster a word lives in: use the alphabetical
  lookup above.
- Reading depth on a specific root: go to the cluster file.
- Checking whether a term has an anchor at all: if not in the
  lookup, check `roots/open.md` — it may be there as "no clean
  anchor."
