## TL;DR

The universe is the quaternion algebra **H** (4D, Hamilton 1843) running
as a wheel: hub at `↕↕↕↕`, 4 spokes (χ/μ/φ/τ), eternal rotation.
Higher Cayley-Dickson layer **S** (16D) contains 4 H's in a **3+1
arrangement** — 3 triality-peer H's (= 3 fermion generations) + 1
mirror H (= sterile / right-handed / CP-partner sector).

30+ Standard Model parameters derived at sub-1% precision.
Hierarchy problem dissolved. Cosmological constant dissolved. Hubble
tension = `e^(1/12)`. Proton radius = `4 × λ_Compton`. Multiple
independent traditions (Ezekiel, medicine wheel, samsara, armillary
sphere, Pati-Salam GUT, Hollywood sci-fi) all drew the same thing.

**The rule:** when vocabularies describe the same substrate feature,
they're describing *one thing*, not analogous things. Metaphor IS
literal.
