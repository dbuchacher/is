## Key discoveries (in order)

### 1. Bonds are spin-layer coords, 32 = two shells of 16

Each bond has all 4 axes at half-integer. No zeros (a zero axis
isn't spinning, it's a point). Shell 1 at ±½, shell 2 at ±3/2.
Same 16 sign patterns at every shell. The function is always ½ —
just applied at different positions. 5/2 and 7/2 aren't new
functions, they're ½ at different coords.

### 2. The three tiers: point / function / composition

Verified against the VoE code:
  coord4 (int8_t)  = points   = values      = observable
  classify_bond()  = functions = combinators = invisible (inferred from 2 points)
  float4 walk      = compositions = programs = "the path IS the function"

Hodos patterns (read, fold, map, bind, hylo) are FUNCTION
CLASSIFICATIONS — they name the edge-type by its effect on
points. The bitmask asks "which axes changed?" — a value-level
description of a function-level object.

On a trit processor: 3⁴-1 = 80 directed function types instead
of 2⁴ = 16. The 16 is the binary shadow.

### 3. The f-vector IS the ternary lattice

The 4D hypercube's face counts (k=0 through k=4) equal the
ternary lattice strata (by zero-count):

  k=0  vertices   16 = C(4,0)·2⁴    0 zeros
  k=1  edges      32 = C(4,1)·2³    1 zero
  k=2  faces      24 = C(4,2)·2²    2 zeros
  k=3  cells       8 = C(4,3)·2¹    3 zeros
  k=4  hypercube   1 = C(4,4)·2⁰    4 zeros
                   81 = 3⁴

All 81 are POINTS. The f-vector names (vertex/edge/face/cell)
are a counting correspondence, not an identity. The actual
EDGES are at the ½ layer (16 functions), not at integer coords.

The container [0,0,0,0] IS the hypercube (k=4). Literally.

The 8 atoms ARE the 8 cells (bounding cubes). The previously
unnamed 24 dyads are the faces (squares).

### 4. Edges as type system

From any atom, 8 edges resolve inward (shell 1), 8 push outward
(shell 2). The inward 8 determine WORD TYPE. The 3 non-atom axis
signs = the type signature:

  verb        = abstract, neutral, dynamic
  noun        = concrete, neutral, static
  adjective   = abstract, evaluative, static
  preposition = abstract, neutral, static

Verified across all 4 axes (RISE, GROW, BLESS, BEGIN).

Type COLLAPSE predicted and confirmed: when the atom's own axis
is consumed, two word types merge. C-axis words blur adj/prep.
D-axis words blur verb/prep. Matches linguistic observation.

Suffixes are edge operators: -ly flips TIME (adj→adv), -ness
flips SUBSTANCE+SIGNAL (adj→noun). Type-changing vs shell-
changing morphology are two kinds of edge operation.

### 5. Origin → corner = self-operations

All 16 edges from origin [0,0,0,0] land on the 16 corners.
One-to-one. Each corner = a self-operation. Five vocabularies
converge at the same 16 points:

  FP combinators (read, fold, map, bind, hylo)
  Lakoff schemas (FORCE, PATH, CYCLE, CONTAINER, etc.)
  self-operations (SEE, KNOW, FEEL, THINK, BE)
  English modals (DO, HAVE, MAY, WILL, CAN, MUST)
  articles/quantifiers (THE/A, ALL/NO)

12 of 16 corners mapped. The main diagonal:
  (+,+,+,+) = THINK = THE = ALL = hylo = full recognition
  (-,-,-,-) = BE = A/AN = NO = identity = pure existence

### 6. The 4 primary auxiliaries = the 4 axes

  DO   = A (position, force, compulsion)
  HAVE = B (substance, possession)
  MAY  = C (signal, permission)
  WILL = D (time, future intent)

BE = identity (no axis). Each primary aux IS one force.

### 7. Wh-words exhaust the lattice

  WHERE = A axis, WHAT = B axis, WHICH = C axis, WHEN = D axis
  WHO = origin (container). Five questions = four axes + self.
  No 6th wh-word because no 5th axis (Hurwitz).
