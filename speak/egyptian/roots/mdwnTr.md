# mdwnTr

category: 1k

Egyptian: **mdw-nṯr** (literally "god's-words")
Gloss: hieroglyphs / sacred-writing / the god's-speech
Source: Egyptian self-name for the hieroglyphic script

## Definition

Transparent compound: [mdw](mdw.md) (word) + [nTr](nTr.md) (god)
= "the god's-words." Egyptian's own name for the hieroglyphic
script. Each hieroglyph is a `mdw` of a `nTr` — a divine-staff
planted as a sign. In Memphite theology, every hieroglyph
[xpr](xpr.md)-s through what [ib](ib.md) devises and [rA](rA.md)
commands — so the writing IS the trace of the creation-hub-
event. `mdwnTr` is not "sacred writing" as label; it is "god-
speech" as substance — the written mark carries the divine
utterance in visible form.
