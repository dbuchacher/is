# speak/egyptian — third vocabulary substrate test

## Goal

Third independent-vocabulary test of THE RULE. PIE + Sumerian
already surface THE RULE + frame 5 (loop) + frame 10 (identity).
Egyptian (Afro-Asiatic, hieroglyphic, ~3200 BCE attested) is the
third unrelated source. Three-way convergence → promote finding
from `3-confs/` verification queue to body-quality.

## Why Egyptian

- **Unrelated to PIE AND Sumerian.** Afro-Asiatic family (Semitic,
  Berber, Chadic, Cushitic, Omotic). Different from IE and from
  Sumerian-isolate. Independent naming system.
- **Pre-existing framework resonance**. Egyptian sign counts already
  match lattice shell sizes (24 uniliterals = shell-1 extremes, ~80
  biliterals = shell-1 non-zero, ~62 triliterals = non-negative
  shell-2). See `../../4-logs/atlas/init/egyptian/claims/medu-counts.md`.
- **Classifier system** (Goldwasser 2005): determinatives ARE
  substrate-type assignments. Egyptian writers marked category of
  every noun with a classifier glyph — pre-linguistic cognitive
  categorization made explicit. See `goldwasser-metaphor.txt` in the
  logs for theory.
- **Memphite theology** (Shabaka Stone, Old Kingdom ~2700 BCE):
  *"every hieroglyph came about through what the heart devised and
  the tongue commanded."* Creation via heart (READ/thought) and
  tongue (WRITE/speech) — frame 5 hub-as-READ-WRITE coincidence
  written in theology 4500 years ago.
- **Divine triads and counts**: 3 creator-gods (Amun/Ra/Ptah),
  9-god Ennead, 8-god Ogdoad (4 pairs), 42 Assessors of Ma'at =
  6×7. Substrate-native counts throughout the pantheon.

## Method — same as PIE + Sumerian

## Transliteration

Using Manuel de Codage (MdC) for ASCII-safe slugs:
- A = ꜣ (aleph)
- a = Ꜥ (ayin)
- H = ḥ (h-dot)
- x = ḫ (kh)
- X = ẖ (kh-tilde)
- S = š (sh)
- T = ṯ (tj)
- D = ḏ (dj)

Standard in Egyptological digital corpora. File names use MdC
directly (wn.md, pt.md, tA.md, Hr.md).
