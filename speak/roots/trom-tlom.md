# trom-tlom

category: 4b  | morpheme: suffix

PIE: *-trom / *-tlom
Gloss: instrument / place suffix
Descendants: -tor, -trum, theater, arbitrium

## Definition

Appends to a verb-root and names the tool or the spot by which
the act is done — the thing-that-bears-the-verb rather than the
body-that-does-it. Where [bher](bher.md) names the carrying,
`bher` + trom-tlom names the carrying-thing: the pole, the
basket, the means. Where [h1ey](h1ey.md)-like going happens,
the suffix names the path-thing. Sits next to
[ter-tor](ter-tor.md) as its mirror: ter-tor gives the agent
(who does), trom-tlom gives the means (what-through the doing
happens). Greek *theatron* is "seeing-place" built this way;
Latin *aratrum* is "plow" as the plowing-thing.

## Composes-with examples

- [bher](bher.md) + trom-tlom → *bhertrom* "that by which one
  carries, a bearing-pole"
- [h1ey](h1ey.md) + trom-tlom → *h1eytrom* "going-place, path-
  thing"
- [wekw](wekw.md) + trom-tlom → "speaking-place" (theater-like)
