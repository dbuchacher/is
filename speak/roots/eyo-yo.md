# eyo-yo

category: 4b  | morpheme: suffix

PIE: *-eyo- / *-yo-
Gloss: adjectival / material suffix
Descendants: -y, -ine, Latin *-ius*

## Definition

Appends to a noun-root and marks made-of — the word built this
way names the stuff-quality of the stem. Where a root names
gold, root + eyo-yo names golden: the thing whose body is
gold-matter. Where a root names stone, root + eyo-yo names
stony. The suffix is the grammar's way of marking material
belonging: harder than [iko](iko.md)'s pertaining-to, this
says the thing is actually the stem's stuff. English -y
(stony, earthy) and -ine (bovine, divine) carry it forward;
Latin *-ius* and Greek *-eos* ride the same pivot.

## Composes-with examples

- gold-root + eyo-yo → *golden* "made-of-gold"
- stone-root + eyo-yo → *stony*
- [deiwos](deiwos.md) + eyo-yo → *divine* "made-of-god-stuff"
