# yos

category: 4b  | morpheme: suffix

PIE: *-yos
Gloss: comparative suffix
Descendants: -er (bigger), Latin *-ior*

## Definition

Appends to an adjective-root and marks more-than — the speaker
holds two things and says one stands further out along the
same line. Where a root names "big," root + yos names "bigger":
the same quality held but tilted, one body further up the
scale than the other. The suffix is the grammar's way of
doing the pair-gesture: [h1eg](h1eg.md) points at one, points
at the other, and the yos-word carries the tilt. English -er
(taller, bigger, older), Latin *-ior*, Greek *-iōn* all ride
this single pivot. Pairs directly with [isto](isto.md) which
takes the tilt all the way to the far end.

## Composes-with examples

- big-root + yos → *bigger* "more-big-than-this-one"
- old-root + yos → *older*
- tall-root + yos → *taller*
