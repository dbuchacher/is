# seeds — the grounding vocabulary

33 Paleolithic PIE roots chosen as the irreducible starting set.
These are the ONLY roots allowed to ground in deixis (pointing at
world) without defining themselves via other roots.

Every other Layer 1 root (150 of them) must define in terms of
these seeds + already-defined roots.

## Rationale

Selection principle: pick roots that are (a) maximally fundamental
in the Paleolithic pre-agricultural vocabulary, (b) categorically
diverse (cover pointing, being, action, motion, perception, speech,
spatial direction), (c) irreducible in the sense that defining them
"using other words" forces circularity or requires external deixis
anyway.

The kernel hypothesis: if the wheel framework's THE RULE holds,
these 33 seeds will themselves cluster on substrate features:
- 1a pronouns → the deictic layer (observer-at-hub pointing)
- 1h being/action → trit values + write-side operations
- 1i motion → spin / force / walk primitives
- 1j perception → READ force + cognition
- 1k speech → CALL force
- 1n spatial → the 4 axes manifesting as direction

## Deictic seed convention

The 9 deictic seeds (1a) are marked with `**seed**` and `**deictic**`
tags in their atomic files. Their definition section says "grounds
in pointing" — they don't reference other roots.

The 24 non-deictic seeds are marked `**seed**` only. Their definitions
may reference other seeds or ground in pointing. They carry the
structural load of the kernel.

## Falsification

If we can't close the 150 non-seed roots using only this 33-root
seed set + previously-defined roots, the seed set needs expansion.
If we need to add >10 more seeds to close, the "minimum kernel"
hypothesis is weakened. If the final dependency graph doesn't
cluster on wheel substrate features, the emergence prediction
fails.
