# structure — one, order, law

Roots naming the substrate's self-coherence: one-ness (THE RULE),
ordered arrangement (cosmos), cosmic order / law (ṛta). These are
how PIE and daughter languages named "there's a single coherent
substrate underneath" — exactly the framework's THE RULE.

---

### *oi-no-*

"one, unique"

```
  → one, unus (Latin), unit, unity, union, only, alone, atom
    (a-tomos = not-divisible, from Greek, but the one-ness is
    there)
  → Greek oinos, Sanskrit eka (related via PIE branches)
  → Germanic: one, ein, ēn
  → THE RULE: one substrate, not two
```

Paired with *wer-* to give **universe** = *unus-versus* =
one-turned = one-unified-by-rotation. The two deepest roots combine
to name what the framework calls the wheel.

### *kosmos*

Greek: "order, arrangement, ornament"

```
  → cosmos, cosmetic, cosmopolitan
  → Sanskrit analogous concept in *ṛta* (order)
  → cosmos = one ordered arrangement = one H
```

Greek *kosmos* is narrower than PIE — it's specifically Greek —
but crucial for our naming: **cosmos = one H at H-scope**. The
universe (= wheel) contains multiple cosmoses (3 peer + 1 mirror
at S-scope).

Note the root's dual meaning: "order" (arrangement) and "ornament"
(cosmetic). An ordered structure IS beautiful; decoration follows
from arrangement. Same substrate fact: coherent ordering is what
makes a thing appear as a unified object.

### *ṛta* (*h₂er-*)

Sanskrit: "cosmic order, truth"

```
  → Avestan asha (same concept)
  → PIE root *h₂er-* "to fit together, join"
  → Greek harmonia, arthron (joint), arithmos (number, order)
  → Latin ars, artus (joint), ordo (order, arrangement)
  → the frame's self-consistency — THE RULE made real
  → arguably the non-Greek equivalent of kosmos
```

**Ṛta** and **kosmos** name the same substrate fact in different
language traditions. Vedic tradition says the universe runs on
*ṛta*; Greek tradition says the universe is *kosmos*. Both are
naming the frame's self-consistency — the substrate coheres
because one set of rules covers everything.

Note: PIE *h₂er-* gives both "fit together" (art, arthron, armor)
and "number / order" (arithmos, ordo). Numbers ARE arrangements;
arrangements ARE joints that fit. Same coord.
