# Petrie Measurements — Tier 1: Primary Dimensions

The numbers encoded in stone: exterior dimensions, chamber sizes, coffer.

## Provenance

Source: W.M.F. Petrie, "The Pyramids and Temples of Gizeh" (1883)
epub: ~/info/pdf/petrie-pyramids-gizeh-1883.epub (scanned page images)
Full OCR: petrie-ocr.txt (7004 lines, tesseract on 157 page images)
Full extraction: petrie-measurements.md (935 lines, 657 entries)
Split: tier1 (this file), tier2 (structure), tier3 (context)

## Processing

1. OCR'd all 157 page images with tesseract
2. Agent extracted every number + context into petrie-measurements.md
3. Converted all inch measurements to royal cubits (1 cubit = 20.632 in)
4. Where cubits snapped to clean values (±0.05): inches stripped, designed
   cubit value kept. These are resolved — the builders thought in cubits.
5. Where cubits DON'T snap: raw inches + cubits retained. These are open
   questions — measurement error, construction tolerance, or undecoded.
6. Angles, weights, counts: kept as-is (not cubit-based)

## Reading this file

- Rows with `designed_cubits` only: the intent is known. 20 means 20 cubits.
- Rows with `value | unit | cubits`: doesn't snap. Worth investigating.
- Angles, counts, weights: original units, no conversion.
