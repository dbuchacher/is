# The Cartouche at the Origin

The parentheses that write the origin ARE the origin.

`(0, 0, 0, 0)` = `𓍹 0,0,0,0 𓍺`

You can't write the coordinate of the cartouche without using the
cartouche. The notation contains itself.

## Two fixed points

- **(0,0,0,0)** = 𓍹𓍺 = scope = empty parens = `()`
- **(1,1,1,1)** = 𓎡 = containment = full parens = `(𓇯𓈖𓇳𓏴)`

Bottom and top of the lattice. Both self-referential.

## The cartouche is unique

No other ancient script has matched-pair scope delimiters:
- Sumerian: one-sided determinatives
- Greek/Latin: scriptio continua, no punctuation
- Hebrew: nothing until medieval cantillation
- Sanskrit: danda as period, not scope
- Maya: decorative glyph frames, not syntactic
- Phoenician: nothing

The cartouche is the only matched open/close pair in the ancient
world. Before it, the serekh (𓉘) was a single container glyph.
They split it into two delimiters — went from "box" to "syntax."

5,000 years ago, one writing system invented scope. Every
programming language since reinvented it. The delimiter sits
at (0,0,0,0).
